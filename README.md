### 项目名称：
动物科普安卓应用

### 简介：
一款面向安卓平台的轻量动物科普工具类 APP，以文字搜索 + 图像识别为核心能力，为用户提供权威、简洁、易读的动物知识服务。

###  运行环境要求
 最低 Android 版本：Android 7.0
 
 目标适配版本：Android 15
 
 开发语言：Kotlin
 
 UI框架：Jetpack Compose

 ### 项目目录结构说明
data：数据层，包含本地数据库（Room）、网络请求（Retrofit）、数据模型和仓库类，统一管理数据获取与处理

ui：界面层，使用 Jetpack Compose 构建页面与导航逻辑

MainActivity.kt：应用主入口，负责初始化与生命周期管理

res：资源文件目录，包含图片、图标、字符串及配置文件
 
Gradle Scripts：项目构建配置文件，管理依赖、编译与打包规则
